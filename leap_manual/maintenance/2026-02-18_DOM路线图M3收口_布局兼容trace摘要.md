## 能力映射（当前脚本形态）

### 1. Layout / 基础行为（`dom-m3-minimal`）

`tests/scripts/integration/test-leapenv-dom-m3-minimal.js` 覆盖了 M3 阶段常见组合能力，包括：

- sibling / children / document.children 关系
- `offsetParent` / `offsetLeft` / `offsetTop`
- `offsetWidth` / `clientWidth` / `boxSizing`
- `display:none` 下尺寸行为
- `textContent` 写入与 `childNodes` 清空
- `snapshotNodeForTrace()` / `traceFirstDiff()` 的正反例

### 2. 兼容与句柄安全（`dom-handle-guard` / `dom-native-ssot`）

`tests/scripts/integration/test-leapenv-dom-handle-guard.js` 与 `test-leapenv-dom-native-ssot-consistency.js` 提供了 native DOM 侧的关键约束验证：

- 同一 handle wrapper 缓存复用
- 文档释放后 stale handle 读写防护
- JS 侧与 native 侧布局结果一致性（SSOT consistency）
- 删除节点后 native snapshot 同步更新

### 3. Trace 能力（`dom-native-trace`）

`tests/scripts/integration/test-leapenv-dom-native-trace.js` 验证：

- native snapshot 自比对匹配
- 故意修改期望后 `traceFirstDiff()` 能输出首个差异路径

这对应历史条目中的 “trace” 能力收口意图。
